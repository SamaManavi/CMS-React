# 📘 CMS Admin Dashboard (React)

A modern and responsive **Content Management System (CMS)** dashboard
built with React.\
This project includes full CRUD functionality, theme switching
(Dark/Light mode), and a clean admin UI for managing users and courses.

------------------------------------------------------------------------

## 🚀 Features

-   📊 Admin Dashboard Overview
-   👤 User Management (Create, Read, Update, Delete)
-   🎓 Course Management (Create, Read, Update, Delete)
-   🌗 Dark / Light Mode Toggle
-   ✅ Form Validation using Zod
-   🔥 Toast Notifications
-   📡 Mock REST API using JSON Server
-   🎨 Styled with TailwindCSS

------------------------------------------------------------------------

## 🛠 Tech Stack

-   React 19
-   Vite
-   React Router v7
-   TailwindCSS v4
-   Zod (Schema Validation)
-   React Hot Toast
-   React Icons
-   JSON Server (Mock Backend)
-   ESLint

---------------------------------------------------------------


## ⚙️ Installation

``` bash
git clone https://github.com/your-username/cms-react.git
cd cms-react
npm install
```

------------------------------------------------------------------------

## ▶️ Run Development Server

``` bash
npm run dev
```

------------------------------------------------------------------------

## 🗄 Run Mock Backend (JSON Server)

``` bash
npm run server
```

Server runs on: http://localhost:5173

------------------------------------------------------------------------

## 🧠 Form Validation

All forms use **Zod schema validation** to ensure: - Required fields -
Proper input formats - Clean error handling

------------------------------------------------------------------------

## 🌗 Theme Mode

The application supports dynamic **Dark and Light themes** using
TailwindCSS utilities.
